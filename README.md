Angularjs-Bootstrap-Faye-Express-Chatroom
=========================================

A template chatroom project with AngularJS + Bootstrap + Faye + ExpressJS (Node.JS) 

## How to use
```
npm install
bower install
node server.js
```

## Version
### Frontend packages
```
    "angular-strap": "~2.1.0",
    "angular-faye": "~0.2.1",
    "angularjs-scroll-glue": "~0.0.1"
```

### Backend packages
```
    "body-parser": "^1.8.0",
    "express": "^4.8.8",
    "faye": "^1.0.3"
```
